SRC Container API - Outline

Updated 6.29.23

This page is meant to be a very crude help page outlining of the endpoints this API provides. Thorough documentation of each endpoint can be found on Github at the link below.

http://github.com/JaxsonP/src-container-api

Start App

Starts the specified app

Usage: http://placeholder.url/[FACILITY_ID]/start-app?name=[APP_NAME]

Stop App

Stops the specified app, forcefully killing it if it doesn't stop after 10 seconds

Usage: http://placeholder.url/[FACILITY_ID]/stop-app?name=[APP_NAME]

Pause App

Pauses the specified app. The app must be running in order to pause

Usage: http://placeholder.url/[FACILITY_ID]/pause-app?name=[APP_NAME]

Unpause App

Unpauses the specified app. The app must be paused in order to unpause

Usage: http://placeholder.url/[FACILITY_ID]/unpause-app?name=[APP_NAME]

Restart App

Restarts the specified app.

Usage: http://placeholder.url/[FACILITY_ID]/restart-app?name=[APP_NAME]

Kill App

Forcefully kills the specified app.

Usage: http://placeholder.url/[FACILITY_ID]/kill-app?name=[APP_NAME]

Create App

Creates an app using the specified image.

Usage: http://placeholder.url/[FACILITY_ID]/create-app?image=[IMAGE_NAME]&user=[USER_NAME]

Delete App

Deletes the specified app, removing its data.

Usage: http://placeholder.url/[FACILITY_ID]/delete-app?name=[APP_NAME]

Hard Reset App

Resets the specified app, removing its data.

Usage: http://placeholder.url/[FACILITY_ID]/hard-reset-app?name=[APP_NAME]

Get App Names

Returns an array of all apps on the specified SRC whether running or not.

Usage: http://placeholder.url/[FACILITY_ID]/get-app-names

Get App Status

Returns basic status information about an app.

Usage: http://placeholder.url/[FACILITY_ID]/get-app-status?name=[APP_NAME]

Get App Stats

Returns lower-level hardware information about an app.

Usage: http://placeholder.url/[FACILITY_ID]/get-app-stats?name=[APP_NAME]

Get App Info

Return more detailed information about an app.

Usage: http://placeholder.url/[FACILITY_ID]/get-app-info?name=[APP_NAME]

Get Uptime Summary

Returns log data about the uptime of an app, over the specified duration (hour, day, week, or month)

Usage: http://placeholder.url/[FACILITY_ID]/get-uptime-summary?name=[APP_NAME]&duration=[DURATION]

Request Image

Requests an image to be "verified" and pulled from Docker Hub.

Usage: http://placeholder.url/[FACILITY_ID]/request-image?image=[IMAGE_NAME]

Get Images

Returns an array of JSON objects representing all existing images on the specified SRC and their data.

Usage: http://placeholder.url/[FACILITY_ID]/get-images

Swarm Create App

Creates an app on a swarm network using the specified image.

Usage: http://placeholder.url/[FACILITY_ID]/swarm-create-app?image=[IMAGE_NAME]&user=[USER_NAME]

Swarm Kill App

Forcefully kills an app on a swarm network.

Usage: http://placeholder.url/[FACILITY_ID]/swarm-kill-app?name=[APP_NAME]

Swarm Get App Names

Returns an array of all apps on the specified swarm-mode SRC whether running or not.

Usage: http://placeholder.url/[FACILITY_ID]/swarm-get-app-names

Swarm Get App Status

Returns basic status information about an app on a swarm network

Usage: http://placeholder.url/[FACILITY_ID]/swarm-get-app-status?name=[APP_NAME]

Swarm Get App Stats

Returns lower-level hardware information about an app on a swarm network.

Usage: http://placeholder.url/[FACILITY_ID]/swarm-get-app-status?name=[APP_NAME]

Swarm Get App Info

Returns more detailed information about an app on a swarm network.

Usage: http://placeholder.url/[FACILITY_ID]/swarm-get-app-status?name=[APP_NAME]

Swarm Get Node Names

Returns an array of the hostnames of the nodes in a swarm network.

Usage: http://placeholder.url/[FACILITY_ID]/swarm-get-node-names

Swarm Get Node Status

Returns basic status info about a node in a swarm network.

Usage: http://placeholder.url/[FACILITY_ID]/swarm-get-node-status?hostname={{hostName}}

Swarm Get Node Info

Returns detailed lower-level info about a node in a swarm network.

Usage: http://placeholder.url/[FACILITY_ID]/swarm-get-node-info?hostname={{hostName}}
